Axon Framework

Axon von der Firma AxionIQ ist ein „CQRS Framework für skalierbare, hochperformante Java Anwendungen“. Es implementiert eine Architektur basierend auf dem CQRS Muster und stellt für dessen Umsetzung vorgefertigte Abstraktionen zur Verfügung.

Reactive Backend

Covers the creation of the Spring Boot application using the required dependencies and the development of the backend layers, paying special attention to the Controllers with WebFlux and the Cassandra reactive driver.

Reactive Database

Cassandra ist ein einfaches, verteiltes Datenbankverwaltungssystem für sehr große strukturierte Datenbanken. Es ist auf hohe Skalierbarkeit und Ausfallsicherheit bei großen, verteilten Systemen ausgelegt.
